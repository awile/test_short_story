# Vertical Short Story Lab

This repo is a small workspace for creating a vertical short-form video story while testing AI-assisted media tools.

It keeps story planning, generated assets, experiments, and reusable Python code organized so the process can eventually become a tool for helping others make short video stories.

## Structure

- `story/`: premise, characters, script, and shot list
- `assets/`: references, generated media, and final exports
- `experiments/`: quick tests for image, video, voice, and pipeline ideas
- `src/`: reusable Python code for tools and pipelines
